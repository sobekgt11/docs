---
title: Listar Templates
description: Retorna uma lista paginada de templates com filtros de busca e data de atualização.
openapi: 'GET /api/templates'
---

<RequestExample>
<CodeGroup>

```bash cURL
curl -G "https://api.formsign.com.br/api/templates" \
  -H "Authorization: Bearer <TOKEN>" \
  --data-urlencode "page=1" --data-urlencode "pageSize=10"
```

```javascript JavaScript
await fetch('https://api.formsign.com.br/api/templates?page=1&pageSize=10',{
  headers:{'Authorization':'Bearer <TOKEN>'}
});
```


</CodeGroup>
</RequestExample>

### Parâmetros de Query
<ParamField query="page" type="integer" default="1">Número da página.</ParamField>
<ParamField query="pageSize" type="integer" default="10">Tamanho da página (máx 100).</ParamField>
<ParamField query="searchText" type="string">Texto para busca.</ParamField>
<ParamField query="lastUpdateStartDate" type="string">Filtro por data (início).</ParamField>
<ParamField query="lastUpdateEndDate" type="string">Filtro por data (fim).</ParamField>

### Respostas
<ResponseExample>
```json 200
{ "page":1, "pageSize":10, "totalPages":2, "totalRecords":12, "templates":[{"templateId":"tpl_123","submissionType":"document"}] }
```
</ResponseExample>
