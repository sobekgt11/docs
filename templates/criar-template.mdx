---
title: Criar Template
description: Cria um template de documento ou formulário para reutilização.
openapi: 'POST /api/templates'
---

Este endpoint permite criar templates reutilizáveis para documentos ou formulários. Para documentos, você pode usar uma chave S3 existente ou criar um template de formulário simples.

### Validações

- **submissionType**: Deve ser "document" ou "form"
- **Campos obrigatórios por tipo**:
  - **document**: `s3Key` e `documentSettings` são obrigatórios
  - **form**: `formSettings` é obrigatório
- **fields**: Pelo menos um campo é obrigatório
- **Posicionamento**: Coordenadas x,y,w,h devem ser entre 0-100 (percentuais)

<Note>
**Tipos de Campo**: 
- **signature**: Assinatura digital
- **text**: Texto livre ou validado
- **date**: Seletor de data
- **checkbox**: Múltipla escolha
- **radio**: Escolha única
</Note>

<Tip>
**Reutilização**: Templates criados podem ser usados no endpoint `/api/envelopes/send/template` para envio rápido de envelopes pré-configurados.
</Tip>
