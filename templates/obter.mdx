---
title: Obter Template
description: Retorna os dados completos de um template específico.
openapi: 'GET /api/templates/{templateId}'
---

<RequestExample>
<CodeGroup>

```bash cURL
curl -X GET "https://api.formsign.com.br/api/templates/tpl_123" \
  -H "Authorization: Bearer <TOKEN>"
```

</CodeGroup>
</RequestExample>

### Parâmetros
<ParamField path="templateId" type="string" required>ID do template.</ParamField>

### Respostas
<ResponseExample>
```json 200
{
  "templateId":"tpl_123",
  "templateName":"Contrato PDF",
  "fields":[{"type":"signature","identifier":"assinatura"}],
  "documentSettings": {"documentName":"contrato.pdf"}
}
```

```json 404
{ "error": "Template not found" }
```
</ResponseExample>

### Campos de resposta

<ResponseField name="documentSettings" type="object">
Configurações do documento PDF.
  <Expandable title="documentSettings">
    <ResponseField name="documentName" type="string">Nome do arquivo.</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="fields" type="array">
Campos definidos no template.
  <Expandable title="fields[] (DadosFormulario)">
    <ResponseField name="type" type="string">Tipo do campo.</ResponseField>
    <ResponseField name="identifier" type="string">Identificador.</ResponseField>
  </Expandable>
</ResponseField>
