---
title: Atualizar Template
description: Atualiza campos do template usando o esquema parcial `UpdateTemplateData`.
openapi: 'PUT /api/templates/{templateId}'
---

<CodeGroup>

```bash cURL
curl -X PUT "https://api.formsign.com.br/api/templates/tpl_123" \
  -H "Authorization: Bearer <TOKEN>" -H "Content-Type: application/json" \
  -d '{"templateId":"tpl_123","templateName":"Contrato v2","advancedOptions":{"autoReminders":true}}'
```

</CodeGroup>

### Parâmetros
<ParamField path="templateId" type="string" required>ID do template</ParamField>

### Body (parcial)
<ParamField body="templateName" type="string">Nome do template.</ParamField>
<ParamField body="templateDescription" type="string">Descrição.</ParamField>
<ParamField body="submissionType" type="string">`form` ou `document`.</ParamField>
<ParamField body="fields" type="array">Campos do template.</ParamField>
<ParamField body="advancedOptions" type="object">Opções avançadas (lembretes/expiração).</ParamField>

### Respostas
<ResponseExample>
```json 200
{ "mensagem": "Template atualizado com sucesso", "dados": { "templateId":"tpl_123" } }
```

```json 400
{ "error": "submissionType inválido" }
```
</ResponseExample>
