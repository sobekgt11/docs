---
title: Criar Template
description: Cria um novo template (form ou document) com campos personalizáveis e opções avançadas.
openapi: 'POST /api/templates'
---

<RequestExample>
<CodeGroup>

```bash cURL
curl -X POST "https://api.formsign.com.br/api/templates" \
  -H "Authorization: Bearer <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{
    "templateName":"Contrato PDF",
    "submissionType":"document",
    "documentSettings":{"documentName":"contrato.pdf","fileHash":"<hash>"},
    "s3Key":"uploads/templates/contrato.pdf",
    "fields":[{"type":"signature","identifier":"assinatura","required":true,"x":70,"y":90,"w":20,"h":8,"pageNumber":1}]
  }'
```

```javascript JavaScript
await fetch('https://api.formsign.com.br/api/templates',{
  method:'POST', headers:{'Authorization':'Bearer <TOKEN>','Content-Type':'application/json'},
  body: JSON.stringify({
    templateName:'Contrato PDF', submissionType:'document',
    documentSettings:{documentName:'contrato.pdf', fileHash:'<hash>'},
    s3Key:'uploads/templates/contrato.pdf',
    fields:[{type:'signature', identifier:'assinatura', required:true, x:70, y:90, w:20, h:8, pageNumber:1}]
  })
});
```

```python Python
import requests
payload={
  "templateName":"Contrato PDF","submissionType":"document",
  "documentSettings":{"documentName":"contrato.pdf","fileHash":"<hash>"},
  "s3Key":"uploads/templates/contrato.pdf",
  "fields":[{"type":"signature","identifier":"assinatura","required":True,"x":70,"y":90,"w":20,"h":8,"pageNumber":1}]
}
r=requests.post('https://api.formsign.com.br/api/templates',json=payload,headers={'Authorization':'Bearer <TOKEN>'})
print(r.json())
```

### Parâmetros (Body)
<ParamField body="documentSettings" type="object" required>Configurações do documento PDF.</ParamField>
<Expandable title="documentSettings">
  <ParamField body="documentSettings.documentName" type="string" required>Nome do arquivo exibido.</ParamField>
  <ParamField body="documentSettings.fileHash" type="string" required>Hash do arquivo para verificação.</ParamField>
  </Expandable>
<ParamField body="s3Key" type="string" required>Chave do arquivo no S3.</ParamField>
<ParamField body="fields" type="array" required>Campos a serem preenchidos/assinados.</ParamField>
<Expandable title="fields[] (DadosFormulario)">
  <ParamField body="fields[].type" type="string" required>Tipo do campo.</ParamField>
  <ParamField body="fields[].identifier" type="string" required>Identificador único.</ParamField>
  <ParamField body="fields[].required" type="boolean" required>Obrigatoriedade.</ParamField>
  <ParamField body="fields[].x" type="number" required>Posição X.</ParamField>
  <ParamField body="fields[].y" type="number" required>Posição Y.</ParamField>
  <ParamField body="fields[].w" type="number" required>Largura.</ParamField>
  <ParamField body="fields[].h" type="number" required>Altura.</ParamField>
  <ParamField body="fields[].pageNumber" type="integer" required>Número da página.</ParamField>
</Expandable>
