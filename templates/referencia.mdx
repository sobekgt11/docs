A API de templates permite criar modelos de documentos enviando o PDF diretamente no formato base64, ideal para integrações server-to-server.

## Operação Disponível:

- POST `/api/templates/base64` — criar template com documento base64

## Características:

- **Upload direto**: Envie o PDF codificado em base64
- **Validação automática**: O sistema valida se é um PDF válido
- **Limite de tamanho**: Máximo 10MB por documento
- **Campos personalizáveis**: Configure campos de assinatura, texto, data, etc.
- **Autenticação**: Requer `Authorization: Bearer <token>`

## Para quem é ideal:

- Integrações server-to-server
- Sistemas que já possuem PDFs em memória
- Automações que não querem gerenciar uploads
- Aplicações que preferem envio direto de dados
