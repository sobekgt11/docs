---
title: Obter Envelope por ID
description: Obtém detalhes completos de um envelope específico.
openapi: 'GET /api/envelopes/{envelopeId}'
---

Este endpoint retorna todos os detalhes de um envelope específico, incluindo campos, destinatários, histórico e configurações.

<RequestExample>
<CodeGroup>

```bash cURL
curl -X GET "https://api.formsign.com.br/api/envelopes/env_abc123456789" \
  -H "Authorization: Bearer <TOKEN>"
```

```javascript JavaScript
const envelopeId = 'env_abc123456789';
const response = await fetch(`https://api.formsign.com.br/api/envelopes/${envelopeId}`, {
  headers: {
    'Authorization': 'Bearer <TOKEN>'
  }
});

const envelope = await response.json();
console.log('Envelope:', envelope.subject);
console.log('Status:', envelope.status);
console.log('Recipients:', envelope.recipients.length);
```

```python Python
import requests

envelope_id = 'env_abc123456789'
response = requests.get(
    f'https://api.formsign.com.br/api/envelopes/{envelope_id}',
    headers={'Authorization': 'Bearer <TOKEN>'}
)

envelope = response.json()
print(f"Envelope: {envelope['subject']}")
print(f"Status: {envelope['status']}")
print(f"Recipients: {len(envelope['recipients'])}")
```

</CodeGroup>
</RequestExample>
