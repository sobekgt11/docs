---
title: Listar Envelopes
description: Lista envelopes com paginação e filtros.
openapi: 'GET /api/envelopes'
---

Este endpoint retorna uma lista paginada de envelopes do usuário, com opções de filtro por status, texto e data.

<RequestExample>
<CodeGroup>

```bash cURL
curl -X GET "https://api.formsign.com.br/api/envelopes?page=1&pageSize=10&status=sent&searchText=contrato" \
  -H "Authorization: Bearer <TOKEN>"
```

```javascript JavaScript
const response = await fetch('https://api.formsign.com.br/api/envelopes?page=1&pageSize=10&status=sent', {
  headers: {
    'Authorization': 'Bearer <TOKEN>'
  }
});

const data = await response.json();
console.log(`Total: ${data.totalRecords} envelopes`);
console.log(`Página ${data.page} de ${data.totalPages}`);
data.envelopes.forEach(env => {
  console.log(`${env.envelopeId}: ${env.subject} - ${env.status}`);
});
```

```python Python
import requests

params = {
    'page': 1,
    'pageSize': 10,
    'status': 'sent',
    'searchText': 'contrato'
}

response = requests.get(
    'https://api.formsign.com.br/api/envelopes',
    params=params,
    headers={'Authorization': 'Bearer <TOKEN>'}
)

data = response.json()
print(f"Total: {data['totalRecords']} envelopes")
for envelope in data['envelopes']:
    print(f"{envelope['envelopeId']}: {envelope['subject']} - {envelope['status']}")
```

</CodeGroup>
</RequestExample>

### Parâmetros (Query)

<ParamField query="page" type="integer" default="1">Número da página (começa em 1).</ParamField>
<ParamField query="pageSize" type="integer" default="10">Quantidade de itens por página (máximo 50).</ParamField>
<ParamField query="status" type="string">Filtrar por status do envelope.</ParamField>
<ParamField query="searchText" type="string">Buscar por texto no assunto ou ID do envelope.</ParamField>
<ParamField query="lastUpdateStartDate" type="string">Filtrar por data de início da última atualização (formato YYYY-MM-DD).</ParamField>
<ParamField query="lastUpdateEndDate" type="string">Filtrar por data de fim da última atualização (formato YYYY-MM-DD).</ParamField>

### Status Disponíveis

| Status | Descrição |
|--------|-----------|
| `draft` | Rascunho (não enviado) |
| `sent` | Enviado para assinatura |
| `completed` | Todas as assinaturas completas |
| `voided` | Cancelado |
| `expired` | Expirado sem completar |
