---
title: Anular Envelope
description: Cancela um envelope, impedindo novas assinaturas.
openapi: 'POST /api/envelopes/void'
---

<CodeGroup>

```bash cURL
curl -X POST "https://api.formsign.com.br/api/envelopes/void" \
  -H "Authorization: Bearer <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{"envelopeId":"env_123","justificativa":"Solicitado pelo remetente"}'
```

```javascript JavaScript
await fetch('https://api.formsign.com.br/api/envelopes/void',{
  method:'POST', headers:{'Authorization':'Bearer <TOKEN>','Content-Type':'application/json'},
  body: JSON.stringify({ envelopeId:'env_123', justificativa:'Solicitado pelo remetente' })
});
```

```python Python
import requests
r=requests.post('https://api.formsign.com.br/api/envelopes/void',json={'envelopeId':'env_123','justificativa':'Solicitado'},headers={'Authorization':'Bearer <TOKEN>'})
print(r.json())
```

### Parâmetros (Body)
<ParamField body="envelopeId" type="string" required>ID do envelope.</ParamField>
<ParamField body="justificativa" type="string" required>Motivo do cancelamento.</ParamField>
