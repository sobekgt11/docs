Esta seção cobre as operações com envelopes:

- GET `/api/envelopes` — listar envelopes
- GET `/api/envelopes/{envelopeId}` — obter envelope
- GET `/api/envelopes/{envelopeId}/history` — histórico
- POST `/api/envelopes/send` — enviar envelope
- POST `/api/envelopes/resend` — reenviar
- POST `/api/envelopes/void` — anular

Requer `Authorization: Bearer <token>`.
