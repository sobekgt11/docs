title: Obter Envelope

openapi: 'GET /api/envelopes/{envelopeId}'
---

### Parâmetros

<ParamField path="envelopeId" type="string" required>
ID do envelope a ser consultado.
</ParamField>

### Respostas

<ResponseExample>
```json 200
{
  "envelopeId": "env_123",
  "status": "pending",
  "subject": "Assine o contrato",
  "recipients": [{"email":"cliente@exemplo.com","name":"Cliente","status":"pending"}],
  "documentSettings": { "documentName": "contrato.pdf" },
  "fields": [{"type":"signature","identifier":"assinatura","x":70,"y":90,"w":20,"h":8,"pageNumber":1}]
}
```

```json 404
{ "error": "Envelope not found" }
```

```json 401
{ "error": "Unauthorized" }
```
</ResponseExample>

### Campos de resposta (principais)

<ResponseField name="envelopeId" type="string" required>
Identificador do envelope.
</ResponseField>
<ResponseField name="status" type="string" required>
Estado atual do envelope (ex.: pending, completed, void).
</ResponseField>
<ResponseField name="recipients" type="array">
Lista de destinatários e respectivos status.
</ResponseField>
<ResponseField name="documentSettings" type="object">
Configurações do documento.
  <Expandable title="documentSettings">
    <ResponseField name="documentName" type="string">Nome do arquivo exibido.</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="fields" type="array">
Campos posicionados no documento.
  <Expandable title="fields[] (DadosFormulario)">
    <ResponseField name="type" type="string">Tipo do campo.</ResponseField>
    <ResponseField name="identifier" type="string">Identificador do campo.</ResponseField>
    <ResponseField name="x" type="number">Posição X.</ResponseField>
    <ResponseField name="y" type="number">Posição Y.</ResponseField>
    <ResponseField name="w" type="number">Largura.</ResponseField>
    <ResponseField name="h" type="number">Altura.</ResponseField>
    <ResponseField name="pageNumber" type="integer">Página.</ResponseField>
  </Expandable>
</ResponseField>
