---
title: Reenviar Envelope
description: Reenvia as notificações de assinatura para os destinatários do envelope.
openapi: 'POST /api/envelopes/resend'
---

<CodeGroup>

```bash cURL
curl -X POST "https://api.formsign.com.br/api/envelopes/resend" \
  -H "Authorization: Bearer <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{"envelopeId":"env_123"}'
```

```javascript JavaScript
await fetch('https://api.formsign.com.br/api/envelopes/resend',{
  method:'POST', headers:{'Authorization':'Bearer <TOKEN>','Content-Type':'application/json'},
  body: JSON.stringify({ envelopeId:'env_123' })
});
```

```python Python
import requests
r=requests.post('https://api.formsign.com.br/api/envelopes/resend',json={'envelopeId':'env_123'},headers={'Authorization':'Bearer <TOKEN>'})
print(r.json())
```
