Bem-vindo à documentação da API do FormSign. Aqui você encontrará guias de autenticação, referências de API e exemplos.

### Como publicar em produção (HTTPS) — caminho mais fácil

1) Mintlify Cloud (recomendado)
- Conecte este repositório no painel do Mintlify.
- Configure o domínio customizado (ex.: `docs.formsign.com.br`). O Mintlify provisiona SSL automaticamente.
- A cada push no branch escolhido, a doc é atualizada.

2) Alternativa: Vercel
- Faça deploy do conteúdo e aponte um CNAME para o projeto. SSL automático.

3) Alternativa on‑premise (NGINX)
- Sirva o build estático atrás de NGINX com Let’s Encrypt.
