Bem-vindo à documentação da API do FormSign. Este guia está organizado por temas para facilitar a navegação:

- Introdução: visão geral e orientações.
- Autenticação: como obter e usar tokens.
- Envelopes: envio, reenvio, histórico e cancelamento.
- Templates: criação, listagem, consulta e atualização.

Use o menu lateral para entrar em cada seção.
